# Pricing Card UI

A modern and responsive **Pricing Card UI** built with front-end technologies to showcase pricing plans in a clean and visually appealing layout. This project is perfect for SaaS websites, landing pages, subscription platforms, and portfolio projects.

---

## 🚀 Features

* Responsive design for all screen sizes
* Modern and clean UI
* Interactive hover effects
* Customizable pricing plans
* Easy-to-read layout
* Reusable and organized code structure

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
